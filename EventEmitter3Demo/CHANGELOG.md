## v4.0.7

1. This library is a third-party library that has been adapted from the EventEmitter3 source repository version 4.0.7 to perform interface validation for the OpenHarmony environment.
2. EventEmitter3 is a high-performance event listener that supports adding event listeners, listening for one-time events, emitting events, removing events, counting the number of event listeners, and listing the names of event listeners.
3. The package management tool has been switched from npm to ohpm.
4. Adapt DevEco Studio:  4.0 Release(4.0.3.513)
5. Adapt SDK: API10 Release(4.0.10.10)
