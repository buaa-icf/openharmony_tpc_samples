# Changelog

## [v1.0.0] 2024-04
- 连接到指定的XMPP服务。
- 断开socket连接。
- 打开流。这个方法会发送流头并开始接收数据。
- 关闭流并断开socket。这个方法会发送流尾并结束连接。
- 重启流。这个方法会重置连接并尝试重新建立流。
- 发送XML元素到服务器。
- 自定义socket连接参数
- 将字符串写入socket。



基于@xmpp/connection 原库0.13.1版本进行适配，使其可以运行在 OpenHarmony，并沿用其现有用法和特性。已支持：
- 连接到指定的XMPP服务。
- 断开socket连接。
- 打开流。这个方法会发送流头并开始接收数据。
- 关闭流并断开socket。这个方法会发送流尾并结束连接。
- 重启流。这个方法会重置连接并尝试重新建立流。
- 发送XML元素到服务器。
- 自定义socket连接参数
- 将字符串写入socket。
