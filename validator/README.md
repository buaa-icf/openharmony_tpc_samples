# validator
## Introduction
This demo shows how to use validator.js, a third-party JavaScript (JS) library that validates and sanitizes strings, on OpenHarmony devices.

## How to Install
```ts
ohpm install validator
ohpm install @types/validator --save-dev // If a syntax error is reported when importing validator, the validator package does not contain the type declaration. You need to download the declaration file **@types/validator** to rectify the syntax error.
```
- For details, see [Installing an OpenHarmony HAR](https://gitee.com/openharmony-tpc/docs/blob/master/OpenHarmony_har_usage.en.md).

## How to Use
```ts
import validator from 'validator';

let result:boolean = validator.isBase32("ZG======")
let result1:boolean = validator.isAscii("foobar")
```
