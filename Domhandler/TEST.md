# text-encoding单元测试用例

单元测试用例覆盖情况

|       接口名       |     参数      |        描述        |是否通过	|备注|
|:---------------:|:-----------:|:----------------:|:---:|:---:|
|      isTag      |  node：节点对象  |    判断节点是不是Tag    |pass   |        |
|     isCDATA     |  node：节点对象  |   判断节点是不是CDATA   | pass |        |
|     isText      |  node：节点对象  |   判断节点是不是Text    | pass |        |
|    isComment    |  node：节点对象  |  判断节点是不是Comment  | pass |        |
|   isDirective   |  node：节点对象  | 判断节点是不是Directive | pass |        |
|   isDocument    |  node：节点对象  | 判断节点是不是Document  | pass |     |
|   hasChildren   |  node：节点对象  |    判断节点是否有子节点              | pass |          |
|    cloneNode    |  node：节点对象  |   克隆节点               | pass |  |
|   firstChild    |      无      |    获取第一个子节点              | pass |          |
|    lastChild    |      无      |    获取最后一个子节点              | pass |          |
|   childNodes    |      无      |     获取所有的子节点             | pass |          |  
|   parentNode    |      无      |      获取父节点            | pass |          |
| previousSibling |      无      |      获取上一个子节点            | pass |          |
|   nextSibling   |      无      |      获取下一个子节点            | pass |          |
|    nodeValue    |      无      |     获取节点值             | pass |          |
|    nodeType     |      无      |     获取节点类型             | pass |          |
|     tagName     | name:Tag的名称 |      获取tag名            | pass |          |
|   attributes    |      无      |      获取所有属性            | pass |          |
