## 1.0.0
- Security authentication: Provides a secure authentication mechanism based on SCRAM (Salted Challenge Response Authentication Mechanism) to ensure the security of user authentication in XMPP communication.
- Hash encryption: This module uses the SHA-1 hash algorithm to encrypt user passwords, enhancing the security of password transmission and preventing passwords from being intercepted and cracked during transmission.
